# Puzzle Upload Guide

This guide explains how to upload puzzles to the Crossword.Network platform.

## Supported Formats

### EclipseCrossword HTML Files
- **Format**: HTML files generated by EclipseCrossword software
- **File Extension**: `.html`
- **Size Limit**: 2MB maximum
- **Features**: Full interactive functionality with progress tracking

## How to Create EclipseCrossword Puzzles

1. **Download EclipseCrossword**
   - Visit [eclipsecrossword.com](http://www.eclipsecrossword.com/)
   - Download the free EclipseCrossword software
   - Install and run the application

2. **Create Your Puzzle**
   - Open EclipseCrossword
   - Create a new crossword puzzle
   - Add words and clues
   - Test your puzzle using the built-in solver

3. **Export as HTML**
   - Go to File → Export → HTML
   - Choose "Interactive HTML" format
   - Save the file with a descriptive name

4. **Upload to Platform**
   - Log in as an admin
   - Navigate to Admin → Puzzles
   - Click "Upload Puzzle"
   - Select your HTML file
   - Fill in metadata (title, description, difficulty, category, tier)
   - Submit the upload

## Upload Requirements

### File Requirements
- **Format**: HTML file (.html extension)
- **Size**: Maximum 2MB
- **Content**: Must contain valid HTML structure
- **Security**: No external resources or suspicious scripts

### Metadata Requirements
- **Title**: Descriptive name for the puzzle (required)
- **Description**: Brief description of the puzzle (optional)
- **Difficulty**: Choose from Easy, Medium, Hard
- **Category**: Select appropriate category
- **Tier**: Choose Free or Premium access level

## Security Validation

The platform automatically validates uploaded files for security:

### Allowed Content
- ✅ EclipseCrossword JavaScript code
- ✅ Inline CSS styles
- ✅ Standard HTML elements
- ✅ Interactive crossword functionality

### Blocked Content
- ❌ External script sources
- ❌ External resource references (images, fonts, etc.)
- ❌ Data exfiltration attempts
- ❌ Suspicious JavaScript patterns
- ❌ Cross-origin requests

## Troubleshooting

### Common Upload Errors

**"Security validation failed"**
- Remove any external script sources
- Ensure all resources are inline
- Check for suspicious JavaScript patterns

**"File size too large"**
- Optimize your HTML file
- Remove unnecessary content
- Maximum size is 2MB

**"Invalid HTML structure"**
- Ensure file contains valid HTML
- Check for proper opening/closing tags
- Verify file is not corrupted

### EclipseCrossword Issues

**Puzzle doesn't load**
- Ensure you exported as "Interactive HTML"
- Check that JavaScript is enabled
- Verify all EclipseCrossword markers are present

**Interactive features not working**
- Confirm you used EclipseCrossword's HTML export
- Check browser console for JavaScript errors
- Ensure puzzle was properly generated

## Best Practices

### File Organization
- Use descriptive filenames
- Include puzzle theme in filename
- Keep files under 1MB when possible

### Content Quality
- Test puzzles before uploading
- Ensure clues are clear and accurate
- Verify all words are valid

### Metadata
- Write clear, descriptive titles
- Provide helpful descriptions
- Choose appropriate difficulty levels
- Select relevant categories
